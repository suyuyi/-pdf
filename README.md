# -pdf
一个可以根据国家自然科学基金服务网上的项目编号爬取相关信息并生成pdf文件的工具。

使用语言为python3.7，可支持多线程同时下载多个文件，不过稳定性还有待验证，可能在线程数过高时会出现问题

详细内容见png_download.py,如果需要直接使用，打包后的exe文件以及依赖库包含在png_download文件夹中，可以下载后使用
